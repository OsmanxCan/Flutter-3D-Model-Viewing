# Flutter 3D Model Viewer

3D model viewing application developed with Flutter.

## Versions

- Flutter: 3.38
- Dart: 3.10

## Current Features

- View `.glb` 3D models
- View `.stl` 3D models
- File picker support
- Window management support

## Planned Features

- 3D object creation
- 3D object editing
- Linux support
- Improved rendering system

## Platform Support

Currently supported only on Windows.

## Flutter Packages

- path_provider
- vector_math
- file_picker
- window_manager

## 3D Libraries

- three_js
- three_js_simple_loaders
- three_js_helpers
